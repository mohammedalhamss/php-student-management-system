[cite_start]A web-based Student Management System built using PHP, MySQL, HTML, and CSS[cite: 11]. [cite_start]It provides a secure administrative dashboard to efficiently manage student records[cite: 12]. 

[cite_start]This system was developed as a project for the Department of Computer Engineering at Istanbul Aydin University[cite: 5], under the supervision of Dr. Öğr. [cite_start]Üyesi Alparslan Horasan[cite: 6].

## 🚀 Key Features

* [cite_start]**Secure Authentication:** Admin login with hashed passwords (`password_verify`) and session management[cite: 13].
* [cite_start]**CRUD Operations:** Full capability to Create, Read, Update, and Delete student records[cite: 15].
* [cite_start]**Real-time Search:** Find students instantly by ID or name using SQL `LIKE` queries[cite: 30].
* [cite_start]**Pagination:** Breaks large datasets into manageable pages (10 records per page) for optimized loading[cite: 28].
* [cite_start]**Dynamic Dashboard:** Real-time statistics displaying total students and administrators, alongside an auto-sliding welcome banner[cite: 26].
* [cite_start]**Intuitive UI/UX:** Features a professional banner [cite: 22][cite_start], dynamic sidebar navigation [cite: 24][cite_start], and breadcrumbs to track user location[cite: 32].

## 🛠️ Technologies Used

* **Backend:** PHP (Procedural with Prepared Statements for security)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Flexbox/Responsive design), Vanilla JavaScript (Slider functionality)
